Usage

Network Module

  • button_pressed: Returns whether MODI Play button is pressed
    • Return value: True if pressed
    • Usage: network.button_pressed

  • dial: Returns MODI Play dial value
    • Return value: 0 ~ 100
    • Usage: network.dial

  • joystick: Returns MODI Play joystick direction
    • Return value: 'up', 'down', 'left', 'right', and None if not pressed
    • Usage: network.joystick

  • left_slider: Returns MODI Play left slider value
    • Return value: 0 ~ 100
    • Usage: network.left_slider

  • right_slider: Returns MODI Play right slider value
    • Return value: 0 ~ 100
    • Usage: network.right_slider

  • timer: Returns if MODI Play timer is up
    • Return value: True / False
    • Usage: network.timer

  • buzzer_on(): Turns on MODI Play buzzer
    • Return value: None
    • Usage: network.buzzer_on()

  • buzzer_off(): Turns off MODI Play buzzer
    • Return value: None
    • Usage: network.buzzer_off()

  • take_picture(): Takes a picture with MODI Play camera
    • Return value: None
    • Usage: network.take_picture()

Button Module

  • How to call Module: button = bundle.buttons[0]

  • button.clicked
    : Returns True when the button is pressed and False when the button is released
    • Return value: True / False
    • Type value: bool

  • button.double_clicked
    : Returns True when the button is double clicked
    • Return value: True / False
    • Type value: bool

  • button.pressed
    : Returns True when the button is pressed
    • Return value: True / False
    • Type value: bool

  • button.toggled
    : Returns True/False each time the button is toggled. It changes the value when the button is pressed.
    • Return value: True / False
    • Type value: bool

Dial Module

  • How to call Module: dial = bundle.dials[0]

  • dial.degree
    : Returns the angle of the dial in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

  • dial.turnspeed
    : Returns the turnspeed of the dial in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

Environment Module

  • How to call Module: env = bundle.envs[0]

  • env.brightness
    : Returns the brightness value in the range of 0 to 100
    • The higher the value, the higher the brightness
    • Return value: 0 ~ 100
    • Type value: float

  • env.humidity
    : Returns the current humidity in the range of 0 to 100
    • The higher the value, the higher the humidity
    • Return value: 0 ~ 100
    • Type value: float

  • env.temperature
    : Returns the current temperature in the range of 0 to 100
    • The higher the value, the higher the temperature
    • Return value: 0 ~ 100
    • Type value: float

  • env.red
    : Returns the red value of the light in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

  • env.green
    : Returns the green value of the light in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

  • env.blue
    : Returns the blue value of the light in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

Gyro Module

  • How to call Module: gyro = bundle.gyros[0]

  • gyro.pitch
    : Returns the pitch value in the range of -180 to 180
    • Return value: -180 ~ 180
    • Type value: float

  • gyro.roll
    : Returns the roll value in the range of -180 to 180
    • Return value: -180 ~ 180
    • Type value: float

  • gyro.yaw
    : Returns the yaw value in the range of -180 to 180
    • Return value: -180 ~ 180
    • Type value: float

  • gyro.angular_vel_x
    : Returns the angular velocity with respect to x axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.angular_vel_y
    : Returns the angular velocity with respect to y axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.angular_vel_z
    : Returns the angular velocity with respect to z axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.acceleration_x
    : Returns the acceleration to x axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.acceleration_y
    : Returns the acceleration to y axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.acceleration_z
    : Returns the acceleration to z axis in the range of -100 to 100
    • Return value: -100 ~ 100
    • Type value: float

  • gyro.vibration
    : Returns the magnitude of vibration in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

Infrared Module

  • How to call Module: ir = bundle.irs[0]

  • ir.proximity
    : Returns the proximity value in the range of 0 to 100
    • ir moudle can measure objects between 4 and 30 cm
    • Return value: 0 ~ 100
    • Type value: float

Mic Module

  • How to call Module: mic = bundle.mics[0]

  • mic.frequency
    : Returns the frequency of the input sound
    • Return value: float value
    • Type value: float

  • mic.volume
    : Returns the volume of the input sound in the range of 0 to 100
    • Return value: 0 ~ 100
    • Type value: float

Ultrasonic Module

  • How to call Module: ultrasonic = bundle.ultrasonics[0]

  • ultrasonic.distance
    : Returns the distance to the object in the range of 0 to 100
    • ultrasonic moudle can measure objects between 50 and 450 cm
    • Return value: 0 ~ 100
    • Type value: float

Display Module

  • text: Sets the text of the display module
    • Return value: current text of the module
    • Usage: display.text = "Hello World"

  • clear(): Clears the screen of display module
    • Return value: None
    • Usage: display.clear()

  • show_variable(): Show a number on the display
    • Return value: None
    • Parameters:
      • variable: number to show on the screen
      • position_x: vertical position of the number
      • position_y: horizontal position of the number
    • Usage: display.show_variable(2.3, 5, 3) # Show 2.3 on position (5, 3) of the screen

Led Module

  • red: Sets the red componenet of the light
    • Return value: red component of the current light
    • Usage: led.red = 255

  • green: Sets the green componenet of the light
    • Return value: green component of the current light
    • Usage: led.green = 255

  • blue: Sets the blue componenet of the light
    • Return value: blue component of the current light
    • Usage: led.blue = 255

  • rgb: Sets the rgb components of the light
    • Return value: Current rgb values
    • Usage: led.rgb = 255, 10, 30

  • turn_on(): Turns on the led to maximum brightness
    • Return value: 255, 255, 255
    • Usage: led.turn_on()

  • turn_off(): Turns off the led
    • Return value: 0, 0, 0
    • Usage: led.turn_off()

Motor Module

  • degree: Sets the degree of motors on both channel
    • Return value: Current degree of the motors
    • Usage: motor.degree = 30, 90

  • first_degree: Sets the degree of motors on channel I
    • Return value: Current degree of the motor on channel I
    • Usage: motor.first_degree = 50

  • second_degree: Sets the degree of motors on channel II
    • Return value: Current degree of the motor on channel II
    • Usage: motor.second_degree = 50

  • speed: Sets the speed of motors on both channel
    • Return value: Current speed of the motors
    • Usage: motor.speed = 50, 100

  • first_speed: Sets the speed of motors on channel I
    • Return value: Current speed of the motor on channel I
    • Usage: motor.first_speed = 50

  • second_speed: Sets the speed of motors on channel II
    • Return value: Current speed of the motor on channel II
    • Usage: motor.second_speed = 50

Speaker Module

  • tune: Sets the frequency and volume of the sound
    • Return value: Current frequency, current volume
    • Usage: speaker.tune = 220, 100 # Plays the sound with frequency 220Hz and volume 100

  • frequency: Sets the frequency of the sound
    • Return value: Current frequency
    • Usage: speaker.frequency = 550

  • volume: Sets the volume of the sound
    • Return value: Current volume
    • Usage: speaker.volume = 50

  • turn_off(): Turns off the sound
    • Return value: 0
    • Usage: speaker.turn_off()

About

PyMODI is a python API for MODI controls. MODI modules are electronic modules that operates using controller area network protocol. You can easily control the electronic modules using PyMODI package. Use PyMODI to make your own MODI creation or copy some of the creations listed above. There are various adaptations of PyMODI in areas such as gaming, machine learning, and education!
